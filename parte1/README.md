## Requisitos

## Arquivos
* `lexer.py`: Código fonte do analisador léxico.
* `teste_correto.lsi`: Exemplo de código válido na linguagem.
* `teste_erro.lsi`: Exemplo de código contendo erro léxico.

## Como Rodar
Para executar o analisador, utilize o comando:

`python3 lexer.py [CAMINHO_ARQUIVO]`

para o teste correto:

`python3 lexer.py ./teste_correto.lsi`

para o teste com erro:

`python3 lexer.py ./teste_erro.lsi`
